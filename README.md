# Add Employers System

## Project Description
This project is a Java application used to manage employers information using a text file.  
The system allows the user to add and store employer data locally without using a database.  
All data is saved and read from a text file.

The project was developed as part of a university assignment to practice Java file handling concepts.

---

## Technologies Used
- Java
- NetBeans IDE
- Text File (.txt)
- File Input / Output

---

## Main Features
- Add employer information
- Save data in a text file
- Read data from the file
- Simple user interface
- No database is used
